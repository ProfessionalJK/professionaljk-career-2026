# Reverse Linked List

This project provides implementations for reversing a linked list in both Python and Java.

## Problem Statement

Given the head of a singly linked list, reverse the list and return the new head.

## Solutions

### Python

- Implements an iterative and recursive approach to reverse a singly linked list.

### Java

- Provides a class-based solution with iterative and recursive methods.

## Usage

Clone the repository and navigate to `algorithms/reverse-linked-list/` to view the source code and run the examples.
